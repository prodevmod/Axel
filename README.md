 ## Components
- HC-SRO4 Ultrasonic Sensor
- Speaker/piezo
- Servo 180 motor
- LCD screen
- Photocell
- 10kΩ potentiometer (for the lcd screen, you can just use an I2C)
- Rgb ring (for a visual effect)
- Breadboard Power Suply for more flexibility
- 2x9v batteries

  ## Connection
- LIGHT_PIN A0   
- TRIG_PIN 5
- ECHO_PIN 4
- SPEAKER_PIN 3
- SERVO_PIN 6

// LCD PINS
- LCD_RS 7
- LCD_EN 8
- LCD_D4 9
- LCD_D5 10
- LCD_D6 11
- LCD_D7 12

//Plus GND and VCC or VDD where needed
